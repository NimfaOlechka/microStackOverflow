MicroStack Overflow:

• Oprette bruger.
• Redigering af bruger.
• Input af spørgsmål
• Redigering af spørgsmål hvis man selv har skrevet det
• Input af svar til spørgsmål
• Redigering af svar til spørgsmål hvis man selv har skrevet det
• Input validering af alle felter ved input inden gemning i database. Form tømmes ikke hvis validering failer.
• Upvoting og downvoting af svar
• Upload af medier
